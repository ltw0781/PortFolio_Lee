# 메인 페이지

이제 '메인 페이지' 디자인을 해줘.
아래 요소를 반드시 포함해야 한다.

[메인 페이지 필수 요소]
 - 상단 네비게이션(로고, 메뉴, 로그인, 장바구니)
 - 히어로 배너
 - 베스트 상품 섹션
 - 최신 상품 섹션
 - 공동구매/이벤트 배너
 - 푸터
 
 [디자인 스타일 옵션]
 - 모던 / 글래스 스타일
 - 애니메이션 약하게 적용

 기존 UI 가이드는 유지하면서, 메인 페이지만 만들어줘.
